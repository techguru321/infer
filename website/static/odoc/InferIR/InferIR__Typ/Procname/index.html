type t =
| Java of Java.t
| C of C.t

| Block of Block.t
| ObjC_Cpp of ObjC_Cpp.t
| WithBlockParameters of t * Block.block_name list

Type of procedure names. WithBlockParameters is used for creating an instantiation of a method that contains block parameters and it's called with concrete blocks. For example: foo(Block block) lock(); bar() foo(my_block) is executed as foo_my_block() my_block(); where foo_my_block is created with WithBlockParameters (foo, my_block)
