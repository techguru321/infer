type debug_level =
| Quiet

(** innocuous, eg emitted once per toplevel execution *)

| Medium

(** still fairly lightweight, eg emitted O(<number of infer processes>) *)

| Verbose

(** go crazy *)

Level of verbosity for debug output. Each level enables all the levels before it.
