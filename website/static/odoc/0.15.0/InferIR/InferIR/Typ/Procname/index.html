type block_name

Type of Objective C block names.

type t =
| Java of Java.t
| C of c
| Linters_dummy_method
| Block of block_name
| ObjC_Cpp of ObjC_Cpp.t
| WithBlockParameters of t * block_name list

Type of procedure names. WithBlockParameters is used for creating an instantiation of a method that contains block parameters and it's called with concrete blocks. For example: foo(Block block) (); bar() (my_block) is executed as foo_my_block() (); where foo_my_block is created with WithBlockParameters (foo, my_block)
